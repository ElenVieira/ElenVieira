## Projeto em Destaque

### Easy-Swap
**Sistema de logística reversa para gestão de trocas em e-commerce**

Projeto desenvolvido com foco em organização operacional, rastreabilidade e melhoria do processo de trocas.

**Principais tecnologias:**
- **Backend:** Supabase / PostgreSQL
- **Frontend:** Next.js
- **Automação:** Fluxos e notificações
- **Diferencial:** apoio operacional com uso de QR Code
